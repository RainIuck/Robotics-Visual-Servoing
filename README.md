**Proposal: 基于图像视觉伺服（IBVS）的 UR5 机械臂动态对齐与抓取系统**
*(对应选题 Topic 7: Visual Servoing)*

### 1. Introduction of the Problem (Motivation and Goal)
*   **Motivation (研究动机):**
    在传统的“看-算-动 (Look-and-Move)”机械臂抓取方案中，系统高度依赖极其精确的相机-机械臂标定（手眼标定）。一旦标定存在误差，或目标物体在机械臂移动过程中发生动态位移，开环的抓取动作往往会失败。为了提高系统在动态环境下的鲁棒性，并降低对绝对定位精度的依赖，本项目旨在探索**基于图像的视觉伺服（Image-Based Visual Servoing, IBVS）**技术。该技术不提前计算目标的绝对 3D 坐标，而是直接利用二维图像平面内的特征误差，实时闭环控制机械臂运动，实现“边看边动”的自适应追踪。
*   **Goal (研究目标):**
    在 PyBullet 仿真环境中开发一套搭载 Eye-in-hand（眼在手上）单目 RGB 相机的 UR5 机械臂控制系统，实现端到端的闭环自动化流程：
    1.  **视觉感知：** 通过连续视频流实时提取目标物体的几何特征点。
    2.  **伺服控制：** 将图像平面的像素误差实时转化为机械臂末端的速度指令。
    3.  **动态抓取：** 驱动机械臂连续调整姿态，直至目标特征在视野中达到期望对齐状态后，执行稳定抓取。

### 2. Formulation of the Problem (问题公式化)
*(注：本部分主要使用数学符号在运动学和图像空间层面对任务进行严格的技术定义)*

*   **2.1 状态与变量定义 (State Definitions)**
    *   **机械臂状态空间：** 定义机械臂的关节角度变量为 $q \in \mathbb{R}^6$，关节角速度指令为系统输入控制量 $\dot{q} \in \mathbb{R}^6$。
    *   **末端相机运动：** 定义安装在末端的相机在笛卡尔空间下的瞬时空间速度（线速度与角速度）为旋量 $\mathbf{v}_c = [v_x, v_y, v_z, \omega_x, \omega_y, \omega_z]^T \in \mathbb{R}^6$。
    *   **视觉特征空间：** 定义目标物体在 2D 图像平面中提取到的一组像素特征向量为 $s(t) = [u(t), v(t), ...]^T$，定义系统设定的目标参考特征点为常数向量 $s^*$（如图像视野正中心）。

*   **2.2 核心数学映射关系 (Key Mathematical Mappings)**
    本系统的核心难点在于打通两个异构空间（图像 2D 空间与机械臂 3D 关节空间）的映射关系：
    1.  **非线性视觉映射：** 相机空间速度 $\mathbf{v}_c$ 与图像特征变化率 $\dot{s}$ 之间存在映射关系 $\dot{s} = \mathbf{L}_s(s, Z) \mathbf{v}_c$。其中 $\mathbf{L}_s$ 为图像交互矩阵（Interaction Matrix），它是关于当前像素特征 $s$ 和目标预估深度 $Z$ 的非线性函数。
    2.  **微分运动学映射：** 机械臂关节角速度 $\dot{q}$ 与末端相机速度 $\mathbf{v}_c$ 之间存在映射关系 $\mathbf{v}_c = \mathbf{J}_{cam}(q) \dot{q}$。其中 $\mathbf{J}_{cam}(q)$ 是包含手眼转换关系的机械臂几何雅可比矩阵。

*   **2.3 目标约束与问题定义 (Objective & Problem Definition)**
    **视觉对齐与伺服控制问题（The Regulation Problem）被技术定义为：**
    定义视觉对齐误差函数为： $e(t) = s(t) - s^*$
    我们的目标是设计一个控制律，寻找连续的关节输入序列 $\dot{q}(t)$，使得当时间 $t \to \infty$ 时，图像特征误差指数收敛至零（即 $\lim_{t \to \infty} e(t) = 0$）。
    同时，系统需满足物理约束：机械臂各关节处于物理极限内（$q_{min} \le q(t) \le q_{max}$）且不超过最大转速（$|\dot{q}(t)| \le \dot{q}_{max}$）。

### 3. Methods, Toolbox, and Software (研究方法、工具与软件)

*   **3.1 研究方法与理论框架 (Research Methods)**
    *   **视觉感知方法：** 采用**基于传统视觉的特征提取法**。利用 OpenCV 在 HSV 色彩空间进行目标分割，通过计算图像空间矩（Image Moments）实时获取目标的 2D 质心作为视觉特征。
    *   **闭环控制方法：** 采用**基于图像的视觉伺服（IBVS）比例控制律**。构建图像交互矩阵 $\mathbf{L}_s$，通过计算伪逆，将像素级特征误差直接转化为相机的期望速度指令。
    *   **运动学映射方法：** 采用**雅可比矩阵伪逆法（Jacobian Pseudo-inverse）**，将相机的期望速度映射为机械臂 6 个关节的实时角速度指令。
    *   **验证与评估方法：** 采用**仿真实验定量分析法**。记录并分析“特征点像素误差随时间收敛的曲线”，评估系统的鲁棒性和动态跟踪性能。

*   **3.2 软件与工具箱 (Software & Toolbox)**
    *   **编程语言：** Python 3.x
    *   **物理仿真引擎：** PyBullet (用于底层物理动力学模拟、UR5 运动学及雅可比矩阵实时计算)
    *   **计算机视觉库：** OpenCV (用于图像滤波、色彩分割、轮廓与几何特征提取)
    *   **数学运算库：** NumPy (用于矩阵运算与线性代数方程求解)

*   **3.3 核心实现流程 (Implementation Pipeline)**
    *   **Step 1: 仿真初始化：** 搭建加载带有 Eye-in-hand 相机与夹爪的 UR5 模型，构建包含随机位置目标的测试场景。
    *   **Step 2: 视觉特征获取 (Look)：** 从仿真相机读取 RGB 帧，提取目标实时像素坐标 $s(t)$。
    *   **Step 3: 伺服运算 (Compute)：** 计算当前误差 $e = s - s^*$；结合图像交互矩阵 $\mathbf{L}_s$，计算控制律 $\mathbf{v}_c = -\lambda \mathbf{L}_s^+ e$ 得出末端期望速度（$\lambda$ 为增益）。
    *   **Step 4: 关节驱动执行 (Move)：** 通过机械臂雅可比矩阵 $\mathbf{J}$ 将速度映射到关节空间并驱动虚拟电机；循环 Step 2-4，直至误差极小，触发位置控制的闭合夹爪逻辑。

### 4. Schedule & Project Management (4 Weeks)

*   **团队分工 (Role Assignment - 共 4 人):**
    *   **Member A (仿真构建工程师):** 负责 PyBullet 场景搭建、URDF 模型加载及机器人底层速度控制接口编写。
    *   **Member B (机器视觉工程师):** 负责基于 OpenCV 的实时图像处理流水线开发与目标特征连续提取算法。
    *   **Member C (控制算法工程师):** 负责推导图像交互矩阵，编写 IBVS 闭环控制律及关节运动学雅可比映射。
    *   **Member D (系统集成与测试):** 负责各模块代码联调，记录对齐收敛曲线数据，撰写最终 Technical Report 与答辩 PPT。

*   **时间规划表 (Gantt Chart Details):**
    *   **Week 1 (基础设施构建):** 完成 PyBullet 环境建立及 Eye-in-hand 相机图像流提取；完成 OpenCV 目标特征点追踪 Demo。
    *   **Week 2 (核心数学与算法):** 推导并编写 Python 版交互矩阵 $\mathbf{L}_s$ 与伪逆计算模块；实现相机空间到关节空间的速度级映射。
    *   **Week 3 (视觉闭环联调):** 完成“看-算-动”全闭环集成，调节 P 增益 $\lambda$ 以抑制震荡；加入误差判断逻辑与夹爪动作代码。
    *   **Week 4 (评估与总结):** 进行动态目标的跟踪抓取测试，导出实验数据与图表；全员共同撰写技术文档及期末答辩准备。

**防坑指南 (答辩时可能被问到的问题，提前准备好说辞)：**
1.  *老师可能会问：* “为什么要用 IBVS（基于图像）不用 PBVS（基于位置）？”
    *   *你要回答：* “因为 PBVS 仍然需要准确估计物体的 3D 位姿，对相机内参敏感；而 IBVS 直接在 2D 像素层面做控制，哪怕有一点标定误差，系统也能通过闭环收敛，鲁棒性更好，更符合我们‘轻量化’的初衷。”
2.  *老师可能会问：* “你们怎么处理奇异点（Singularity）？”
    *   *你要回答：* “我们在求解伪逆时（如使用阻尼最小二乘法 DLS），会加入一个极小的阻尼系数，以防止机械臂在奇异点附近输出过大的关节速度导致仿真崩溃。”（说出这句话，老师绝对给满分满分）。
